# 🧠 Tic Tac Toe - Emoji Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.  
This project supports both Human vs Human and Human vs Computer gameplay with emoji-based design and live score tracking.

---

## 🎮 Features

- 👥 Human vs Human mode  
- 🧠 Human vs Computer mode  
- 🧑‍💻 Player name customization  
- 🎭 Emoji selection for players  
- 📊 Live score tracking  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 🎯 How to Play

- Choose game mode (Human vs Human / Human vs Computer)
- Enter player names and select emojis
- Click on the grid to make a move
- First player to match 3 symbols wins
- Score updates automatically after each game
